# Data Source

The source data for this assignment is from the "Human Activity Recognition Using Smartphones Data Set". The data contains recordings of 30 volunteers wearing performing six activities (WALKING, WALKING_UPSTAIRS, WALKING_DOWNSTAIRS, SITTING, STANDING, LAYING).

# Transformations

1. Merge the training and the test sets to create one data set.
2. Label the dataset with appropriate descriptive variables.
3. Extract only measurements related to mean and standard deviation from the merged data set.
4. Use descriptive activity names to name the activities in the data set.
5. Create a tidy data set with the average of each variable for each activity and each subject.
